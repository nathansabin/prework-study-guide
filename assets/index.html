Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: will save changes
  • git commit -m "added code to HTML file": will add our changes to git hub
  • git pull origin main: will sync our changes with the main branch
